## Contributing
If for some reason you want to contribute to totem, please make a pull request through github and I'll be sure to review it.
You can also open an issue through github's issue tracker and I'll try to get to it as soon as possible.

### Getting Started
Documentation is available at http://totem.mythical83.com/docs. You can also check out the [Getting Started](http://totem.mythical83.com/docs/getting-started) page for help with installing totem and creating a new project

### Installing From Source
To install from source run the following commands:
```bash
clone https://github.com/Mythical84/totem && cd totem
cargo build --release
```

After building find the built binary in the target/release folder
